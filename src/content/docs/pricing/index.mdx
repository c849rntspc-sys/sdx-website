# Pricing Overview

SDX operates on a simple principle: **the platform is free for everyone.** There are no subscription fees, no per-building charges, no API usage costs, and no feature gates.

## Who pays what

| Audience | Cost |
|---|---|
| **Building owners** (data contributors) | Free — no fees to read, write, benchmark, or report |
| **Service providers** (consultants, property managers) | Free — same access as owners when acting on their behalf |
| **Third-party software** (integrators) | Free — no API fees, no per-call charges, no integration costs |
| **Licensed content buyers** (investors, lenders, index providers) | Paid — board-approved data products only |

## The only revenue source

SDX's sole monetisation mechanism is the sale of **licensed content** — aggregate benchmark data, market indices, and anonymised data packs — to third parties who want to consume them. These products are governed by the SDX board and undergo a formal approval process before being offered for sale.

Data contributors receive all licensed content free for their own internal use. Only third parties who want to use the data commercially pay for it.

## No hidden costs

There are no:

- Freemium tiers with locked features
- Per-building or per-metre charges
- API rate limit tiers that require payment to unlock
- Premium support packages
- Data export fees
- Setup or onboarding costs

Every feature described in this documentation is available to every user at no cost.

## Revenue sharing

A portion of licensed content revenue is shared with data contributors through the [Data Dividend](/docs/data-dividend). This means contributors are not just using a free platform — they are earning money from it.
